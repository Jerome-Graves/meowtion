    \textbf{Abstract.} Meowtion is a smart cat collar that infers a cat's health
    from changes in its daily habits. A battery-powered collar classifies behaviour
    (eating, drinking, activity, rest, purring) on the device itself, relays the
    result over Bluetooth Low Energy to a mains-powered base station, and the station
    forwards it to the cloud for a live owner-facing dashboard. The behaviour set is
    not hard-coded: owners define the actions to recognise, label captured clips, and
    a cloud trainer produces tiny quantised models that are delivered back to the
    collar over the air. This document is the end-to-end technical reference for the
    system: hardware, firmware, on-device AI, the training pipeline, the cloud
    backend, the dashboard, the communication protocols, and the security model.
