    \textbf{Abstract.}\enspace Meowtion is a smart cat collar that infers a cat's health from
    changes in its daily habits. Cats instinctively hide illness, so the earliest warning is
    usually a quiet shift in routine, eating less, drinking more, or grooming and moving less,
    rather than an obvious symptom. A battery-powered collar classifies behaviour on the device
    itself with a confidence-gated cascade of two quantised neural networks, relays only the
    result over Bluetooth Low Energy to a mains-powered base station, and the station forwards it
    to a cloud backend that drives a live, owner-facing dashboard with longitudinal health
    analytics. The behaviour set is not hard-coded: owners define the actions to recognise, label
    captured clips, and a cloud trainer produces tiny models that are delivered back to the collar
    over the air. This document is the end-to-end technical reference for the system, covering the
    architecture and data flows, the hardware, the firmware on both microcontrollers, the
    on-device AI, the training pipeline, the cloud backend, the dashboard, the communication
    protocols, and the security and privacy model.
